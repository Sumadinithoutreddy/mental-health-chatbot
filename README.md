# AI Mental Health Chatbot

A web-based AI Mental Health Chatbot built using FastAPI and MongoDB.

## Features
- User Authentication
- Mental Health Chat Support
- AI-based Responses
- MongoDB Database
- REST API
- Swagger Documentation

## Technologies Used
- Python
- FastAPI
- MongoDB
- Uvicorn
- JWT Authentication

## Run Project

```bash
uvicorn app.main:app --reload
```
